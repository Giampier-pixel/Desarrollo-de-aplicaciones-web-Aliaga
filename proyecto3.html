// JavaScript Interactivo
document.addEventListener('DOMContentLoaded', function() {
    // Elementos del DOM
    const interactiveBtn = document.getElementById('interactive-btn');
    const title = document.querySelector('.title');
    
    // Función para cambiar colores dinámicamente
    function changeColors() {
        const colors = ['#667eea', '#764ba2', '#f093fb', '#f5576c'];
        const randomColor = colors[Math.floor(Math.random() * colors.length)];
        title.style.color = randomColor;
    }
    
    // Event Listeners
    interactiveBtn.addEventListener('click', function() {
        changeColors();
        this.style.transform = 'scale(0.95)';
        setTimeout(() => {
            this.style.transform = 'scale(1)';
        }, 150);
    });
    
    // Animación de entrada
    window.addEventListener('load', function() {
        title.style.opacity = '0';
        title.style.transform = 'translateY(-30px)';
        
        setTimeout(() => {
            title.style.transition = 'all 0.8s ease';
            title.style.opacity = '1';
            title.style.transform = 'translateY(0)';
        }, 100);
    });
});

Aprendizajes Clave

Durante el desarrollo de este proyecto de la Semana 2, he profundizado significativamente en las tecnologías fundamentales del desarrollo web frontend. La integración de HTML semántico, CSS moderno y JavaScript interactivo me ha permitido crear una experiencia de usuario coherente y atractiva.

Desafíos Superados
Responsive Design: Implementación de layouts flexibles que se adaptan a múltiples dispositivos
JavaScript Moderno: Uso de ES6+ y manipulación eficiente del DOM
CSS Avanzado: Implementación de gradientes, animaciones y efectos visuales
Optimización: Mejora del rendimiento y tiempos de carga
Reflexión sobre Bootstrap y Tailwind CSS

La implementación de frameworks CSS como Bootstrap y Tailwind CSS ha revolucionado mi approach al desarrollo frontend. Bootstrap proporciona componentes robustos y un sistema de grid confiable, mientras que Tailwind CSS ofrece un enfoque utility-first que permite un control granular del diseño. La combinación de ambos frameworks me ha permitido:

  • Acelerar significativamente el tiempo de desarrollo
  • Mantener consistencia visual en todo el proyecto
  • Implementar diseños responsive sin escribir media queries complejas
  • Crear interfaces modernas con efectos visuales avanzados
