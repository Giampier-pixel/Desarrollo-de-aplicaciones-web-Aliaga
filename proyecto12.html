Semana 12

Desarrollo de un Sistema Web Backend con Django

Temas Aprendidos

Gestión de Formularios Django Admin Middleware Sesiones y Autenticación Validación y Sanitización Campos Calculados

Esta semana profundizamos en el desarrollo backend con Django, aprendiendo sobre gestión avanzada de formularios, personalización del panel administrativo, implementación de middleware personalizado y manejo seguro de sesiones. Exploramos técnicas de validación de datos, sanitización de entrada y creación de campos calculados dinámicos. También vimos cómo Django facilita la creación de aplicaciones web robustas y escalables mediante su arquitectura MVT.

Ejercicios y Resultados

Sistema de registro de usuarios con validación personalizada

from django import forms from django.contrib.auth.models import User from .models import Perfil, Categoria class RegistroForm(forms.ModelForm): nombre = forms.CharField( max_length=100, widget=forms.TextInput(attrs={ 'class': 'form-control', 'placeholder': 'Nombre completo' }) ) email = forms.EmailField( widget=forms.EmailInput(attrs={ 'class': 'form-control', 'placeholder': 'tu@email.com' }) ) class Meta: model = Perfil fields = ['telefono', 'edad', 'categoria', 'mensaje'] def clean_edad(self): edad = self.cleaned_data.get('edad') if edad < 18: raise forms.ValidationError("Debe ser mayor de edad") return edad

3. Middleware y Sesiones

Implementación de middleware personalizado y manejo de sesiones

import logging from django.utils.deprecation import MiddlewareMixin class LoggingMiddleware(MiddlewareMixin): def process_request(self, request): logger.info(f"Request: {request.method} {request.path}") logger.info(f"User: {request.user}") def process_response(self, request, response): logger.info(f"Response: {response.status_code}") return response class SessionMiddleware(MiddlewareMixin): def process_request(self, request): if not request.session.session_key: request.session.create() # Actualizar última actividad request.session['last_activity'] = str(timezone.now()) request.session.modified = True

Reflexión Final

Durante esta semana, al explorar Django como framework backend, me di cuenta de cómo esta herramienta simplifica significativamente el desarrollo web. Al principio, la curva de aprendizaje del patrón MVT y la estructura de Django me resultó desafiante, pero al practicar con formularios, middleware y el sistema de administración, desarrollé una comprensión más profunda de las ventajas de usar un framework bien estructurado.

¿Qué aprendí?

Aprendí a desarrollar aplicaciones web backend robustas con Django, incluyendo gestión de formularios, personalización del admin panel, implementación de middleware y manejo seguro de sesiones y autenticación.

¿Cómo aprendí?

Aprendí mediante la práctica directa con Django, creando formularios interactivos, configurando el panel administrativo y desarrollando middleware personalizado para comprender el flujo de peticiones HTTP.
