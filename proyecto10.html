🐍 Python Basics

Python es un lenguaje de programación interpretado, de alto nivel y con una sintaxis clara y legible que facilita el desarrollo rápido de aplicaciones.

📋 Características Principales

Sintaxis simple, tipado dinámico, orientado a objetos, gran ecosistema de librerías y multiplataforma.

🔧 Paradigmas

Soporta programación imperativa, orientada a objetos, funcional y procedural.

3. Programación Orientada a Objetos
# Clase y herencia class Vehiculo: def __init__(self, marca, modelo): self.marca = marca self.modelo = modelo self.velocidad = 0 def acelerar(self, incremento): self.velocidad += incremento return f"{self.marca} {self.modelo} acelerando a {self.velocidad} km/h" class Coche(Vehiculo): def __init__(self, marca, modelo, puertas): super().__init__(marca, modelo) self.puertas = puertas def describir(self): return f"{self.marca} {self.modelo} con {self.puertas} puertas" # Crear instancias mi_coche = Coche("Toyota", "Corolla", 4) print(mi_coche.describir()) print(mi_coche.acelerar(50))

💭 Reflexión Final

Esta semana me adentré en el mundo de Python, un lenguaje que realmente me ha sorprendido por su simplicidad y elegancia. La filosofía de Python de "código legible" se hace evidente desde el primer momento, donde la indentación no es solo una convención, sino parte fundamental de la sintaxis.
Lo que más me ha llamado la atención es cómo Python logra un equilibrio perfecto entre simplicidad y potencia. Las estructuras de datos nativas como listas, tuplas y diccionarios son increíblemente versátiles y fáciles de usar. La programación orientada a objetos en Python se siente natural y menos verbosa comparada con otros lenguajes.
El manejo de excepciones con try-except proporciona una forma elegante de manejar errores, mientras que el sistema de módulos facilita la organización y reutilización del código. Python demuestra que un lenguaje puede ser tanto potente como accesible para principiantes.
Esta experiencia me ha hecho apreciar cómo diferentes lenguajes de programación pueden abordar los mismos problemas con filosofías distintas. Python, con su enfoque en la legibilidad y productividad, se ha convertido en una herramienta valiosa en mi arsenal de desarrollo.
